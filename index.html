Java is a general-purpose computer programming language that is concurrent, class-based, object-oriented, and specifically designed to have as few implementation dependencies as possible. It is intended to let application developers "write once, run anywhere" (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation.

About Codefucious:
Codefucious was designed to educate you so you can educate others.

Codefucious is usable for all beginner programmers that want to learn the basic aspects of coding. By answering a number of programming questions, not only are you expanding your knowledge of programming theory and increasing your skill, each question answered donates one penny to a scholarship and grant fund for women wanting to gain a STEM education.

Codefucious is completely free for anyone to use.
How it Works:
Codefucious has one goal: to teach.

The website opens and provides a choice for two languages you can learn Python or Java. A short lesson in your selected language will be provided, intermittent with a brief quizzes to test your knowledge. Every lesson learned and every question answered will donate one penny to a scholarship fund, used to provide women in need with a STEM education.

To begin your lesson, simply select a language, and get ready to learn!
About Us
CopyrightPending is a group of students who participated in a group during the Women In Computing (WIC) Hackathon at the Rochester Institute of Technology. They were faced with problem solving in the realm of teaching children how to program or solving a women and gender issue. They decided that they could do both. CopyrightPending is a diverse group of girls, coming from different majors and skill levels, who came together to solve these issues. The product, Codefucious, aims to teach children to code while also helping women advance in the field of technology.
